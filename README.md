# Example Firebase Database

This example shows how to add objects to a Firebase Database.

## Setup

-   Sign up for a Firebase account.
-   : `npm install`
-   Login to firebase: `npm run login`
-   Initialize firebase database: `npm run setup`
    -   When presented with options, select Database and Hosting.
    -   It will generate a number of files to use.
-   Add the database config: `npm run create-config`
-   Push test to hosting: `npm run deploy`

```
# Setup environment
npm install

# Login to Firebase
npm run login

# Initialize Firebase Database and Hosting
npm run setup

# Create the database config
npm run create-config

# Deploy to Firebase Hosting
npm run deploy

# A URL will be displayed that will take you to the click test site
```
